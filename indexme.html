Confidant Compass

Safe and anonymous spaces for teens to share challenges and receive peer guidance.

Concept

Create topic-specific chat rooms where teens can discuss issues like friendships, body image, academic stress, etc., anonymously. Experienced teen moderators ensure respectful discussions and offer guidance.

Novel Functionality

Implement a "Confidant Match" feature that pairs users with peers who have overcome similar challenges, providing authentic and relatable support. Google Search API can be used to find relevant user stories based on keywords or challenges mentioned.

Google Tech

Google Chat API powers secure anonymous chat rooms. Google Cloud Functions can automate the "Confidant Match" feature based on user profiles and chat discussions.
